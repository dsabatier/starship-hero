# Starship Hero

A lightweight Unreal Engine editor plugin that adds a direct shortcut to the built-in Slate Widget Gallery (Slate Surveyor).

## Installation

1. Create a `Plugins` folder in your project's root directory (if it doesn't already exist).
2. Clone or move the `StarshipHero` folder into the `Plugins` directory. 

`git clone https://github.com/dsabatier/starship-hero.git ./StarshipHero`

   ```
   YourProject/
   ├── Content/
   ├── Source/
   ├── Plugins/
   │   └── StarshipHero/
   └── YourProject.uproject
   ```
3. Restart Unreal Engine.
4. If prompted, click **Yes** to rebuild the plugin.

## Usage

1. In the Unreal Editor, click on the **Window** menu in the top toolbar.
2. Look for **Open Slate Demo** (located near the layout options).
3. Click it to launch the **Slate Widget Gallery**.

## Compatibility

- **Unreal Engine**: 5.3+ (Designed for 5.7+)
- **Module Type**: Editor
